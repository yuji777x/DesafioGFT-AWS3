# Implementando minha Primeira Stack com AWS CloudFormation

## Sobre o desafio

Neste laboratório aprendi os conceitos básicos do AWS CloudFormation e como ele facilita a criação de recursos na AWS utilizando Infraestrutura como Código (IaC).

O objetivo foi entender como criar uma Stack e perceber as vantagens de automatizar a infraestrutura ao invés de configurar tudo manualmente.

---

## O que aprendi

Durante as aulas aprendi que o CloudFormation permite criar vários recursos da AWS por meio de um único arquivo de configuração (template).

Também entendi que uma **Stack** representa um conjunto de recursos criados juntos e que podem ser atualizados ou removidos de forma centralizada.

Outro conceito importante foi o de **Infraestrutura como Código (IaC)**, que ajuda a padronizar ambientes, reduzir erros e facilitar a implantação de projetos.

---

## Conclusão

Esse laboratório me ajudou a entender como o AWS CloudFormation automatiza a criação da infraestrutura na nuvem e como esse serviço pode facilitar o gerenciamento de recursos em projetos maiores. Foi um ótimo primeiro contato com Infraestrutura como Código e mostrou a importância da automação no ambiente AWS.
